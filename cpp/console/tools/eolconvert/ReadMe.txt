----------------------------------------------------------------------
End of Line (EOL) Delimter Summary
----------------------------------------------------------------------
1. Unix          :   \n
2. Dos/Windows   : \r\n
3. Mac           :   \r

This tools can convert any one of eol file into another two

Usage:
====================================================================================================
$ EOLConvert   <filename>  --[unix|linux|dos|win|mac]
$ EOLConvert   <filename>  --[unix|linux|dos|win|mac]  --override
====================================================================================================
