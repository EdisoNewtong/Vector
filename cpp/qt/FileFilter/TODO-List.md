# Features 

1. 扩展名 ( 可以是复合扩展名 e.g.     .cpp.backup )
1. 是文件 or 文件夹
1. 文件名 头部/尾部 包含某个/某些 字符
1. 文件包含某些字符 ( 不限 头/尾 ， 中间也可以 )
1. 正则表达式
1. 文件名中， 含有某种字符集的字符 ( 如中文字符，标点符号 ，日文字符 ... )
1. 文件内容中 包含 某个/某些 字符

----------------------------------------------------------------------------------------------------

条件连接  &&   ||   !

条件分组  ( a || b ) && ( c || d )
