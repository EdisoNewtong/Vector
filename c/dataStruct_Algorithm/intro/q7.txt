7. 有以下用 C/C++ 语言描述的算法，说明其功能:

////////////////////////////////////////////////////////////////////////////////////////////////////

//
// 求 幂运算 :  y ^ n
//
void fun(double& y, double x, int n)
{
    y = x;
    while ( n > 1 )
    {
        y = y * x;
        n--;
    }
}
