10. 分析正面程序段中循环语句的执行次数

////////////////////////////////////////////////////////////////////////////////////////////////////

int j = 0, s = 0, n = 100;
do {
    j = j + 1;
    s = s + 10 * j
} while ( j<n && s <n );

////////////////////////////////////////////////////////////////////////////////////////////////////


循环体一共执行了 4 次 :
    1. j = 1;  s = 0  + 10*1 = 10;
    2. j = 2;  s = 10 + 10*2 = 30;
    3. j = 3;  s = 30 + 10*3 = 60;
    4. j = 4;  s = 60 + 10*4 = 100;
